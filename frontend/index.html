            ◎ MODEL FORMULA
          
        
          
        
            
        
              Score = L0_gate × (L1×0.50 + L2×0.30 + L3×0.20)
            
        
            
        
              L0 gateBinary kill switch — regime (VIX) × session window. 0 = no trade.
              L1 (50%)LTQ z-score · OI build velocity · VWAP σ deviation — actual order flow.
              L2 (30%)IV Rank/Pct · GEX proximity · PCR reversion edge — options intelligence.
              L3 (20%)10+/15 heavyweights above VWAP + macro gate 3/4 pass — broad market align.
            
        
            
        
              Why multiplicative? Any layer scoring 0 (e.g. regime = HIGH_VOL) kills the entire trade — prevents entering when even one critical condition fails. Additive models hide bad signals.
